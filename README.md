# Healthcare Appointment No-Show Analysis

## Objective
Analyze healthcare appointment data to identify factors associated with patient no-shows.

## Tools Used
SQL, Power BI, Excel/Python

## Key Questions
- What is the overall no-show rate?
- Which age groups miss appointments most often?
- Does appointment lead time affect no-show behavior?
- Are SMS reminders associated with lower no-show rates?

## Process
1. Cleaned raw appointment data
2. Created calculated fields
3. Queried trends using SQL
4. Built Power BI dashboard
5. Summarized actionable insights
